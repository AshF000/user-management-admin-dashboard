# User Management Admin Dashboard

A modern User Management Admin Dashboard built with React and Vite.

[![Live Preview](https://img.shields.io/badge/Live%20Preview-007fff)](https://user-management-admin-dashboard-ashfiq.vercel.app/)

## Features

- **User List & Search:** View, search, and filter users with debounced input for performance.
- **Pagination:** Pagination component for efficient data navigation.
- **Dark Mode:** Toggle between light and dark themes.
- **Reusable UI Components:** Modular, reusable components (buttons, dialogs, tables, etc.) following atomic design principles.
- **API Integration:** Clean abstraction for API calls using a dedicated `api` layer.
- **Performance Optimizations:** Debounced search, code splitting, and optimized rendering.
- **Modern Tooling:** Built with Vite for fast development and hot module replacement.
- **ESLint & Code Quality:** Linting and code style enforcement for maintainability.

## Tech Stack

- **JavaScript (ES6+)**
- **CSS Modules**
- **React 19+**
- **ESLint**
- **Vite**

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/AshF000/user-management-admin-dashboard.git
   cd user-management-admin-dashboard
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Run the development server:**
   ```sh
   npm run dev
   ```
4. **Open in browser:**
   Visit [http://localhost:5173](http://localhost:5173) to view the app.

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint
